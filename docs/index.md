# **Getting Started**
A file manager for React

## What is useFiles?
useFiles is meant to be a simple hook that exposes a bunch of utility functions that makes dealing with files a lot simpler. 

## Features
Using useFiles give you access to great file utilities such as
- adding files
- removing files
- replacing files
- applying meta data to your files

Those are just naming a few! The best part of this is that you don't need to worry about state managment because useFiles holds that information for you!

## Installation

**Find your prefered package manager.**

**npm:**

```bash
$ npm install usefiles
```

**yarn:**

```bash
$ yarn add usefiles
```
**pnpm:**

```bash
$ pnpm add usefiles
```
**bun:**

```bash
$ bun add usefiles
```
